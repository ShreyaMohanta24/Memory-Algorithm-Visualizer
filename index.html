About the Project

Understanding page replacement in operating systems

💾

Virtual Memory

Virtual memory is a memory management technique that gives programs the illusion of a larger address space than physical RAM permits. Pages are swapped between RAM and disk on demand — making page replacement algorithms critical to performance.

🔄

FIFO Algorithm

First In, First Out replaces the oldest page in memory. Simple to implement but can suffer from Bélády's anomaly — adding frames sometimes increases page faults.

O(n) time · O(f) space
🕰️

LRU Algorithm

Least Recently Used evicts the page that hasn't been accessed for the longest time. It approximates the Optimal algorithm and is widely used in real operating systems.

O(n) time · O(f) space
🎯

Optimal Algorithm

Optimal (OPT/MIN) replaces the page that will not be used for the longest time in the future. Since it requires future knowledge it cannot be implemented in practice — but it provides a theoretical lower bound on page faults.

O(n²) time · O(f) space
